# Todo List

A dark mode todo list app built with HTML, CSS and JavaScript.

## Features
- Add tasks with due dates
- Overdue task detection
- Tasks saved automatically in the browser (localStorage)
- Mark tasks as completed

## How to use
1. Type a task in the input field
2. Optionally pick a due date
3. Press Add or hit Enter
4. Click the checkbox to complete a task
